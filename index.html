Product Hunt CLI

Product Hunt's products right from your terminal.

Product Hunt CLI is a command line tool that allows you to access trending products, latest products, upcoming products, jobs, remote jobs etc from Product Hunt website. It even allows you to search for products right from your terminal.

Prerequisites

1. Node.js
2. npm

Installation

Install npm package

npm install -g product-hunt-cli

Usage

Usage: ph [options] [command]

Options:
-V, --version output the version number
-h, --help output usage information

Commands:
latest|l
upcoming|j
jobs|j
remote|r
topics|t
search|s
author|a
